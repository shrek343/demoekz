# 📝 Как переделать сайт под свою тему

Проект на **HTML + CSS + JS + PHP + MySQL + Bootstrap**.
Файла `config.php` больше нет — **все тексты, картинки и цвета меняются прямо
в файлах**, по месту. В коде стоят пометки `✏️` — это места, которые нужно
заменить на своё.

---

## 1. Тексты (название, описание, карточки, контакты)

| Что поменять | В каком файле | Что искать |
|---|---|---|
| Название компании | `includes/header.php` и `index.php` | «Название вашей компании» |
| Название вкладки браузера | `includes/header.php` | внутри `<title>…</title>` |
| Описание | `index.php` | «Описание вашей экскурсии» |
| Подписи 3 карточек | `index.php` **и** `application.php` | «Название карточки 1/2/3» |
| Контакты (адрес, телефон) | `index.php` | блок «Контакты» |
| Способы оплаты | `index.php` **и** `application.php` | блок «Способы оплаты» |
| Время / статусы | `application.php`, `cabinet.php`, `admin.php` | списки `$times`, `$statuses` |

> ⚠️ Названия карточек, времени и оплаты написаны в **двух местах**
> (на главной и в форме заявки) — поменяй их одинаково в обоих, иначе
> форма не примет выбор. Рядом в коде об этом есть напоминание.

> ⚠️ Первый статус — `Новая`. Его автоматически ставит база данных новой
> заявке, поэтому его лучше не переименовывать (а если переименуешь —
> поменяй и `DEFAULT 'Новая'` в `database.sql`). Средний и последний
> статусы можно называть как угодно.

---

## 2. Картинки (логотип и фото карточек)

В папке `img/` лежат **картинки-заглушки**. Замени их своими файлами
из папки с исходниками (имена файлов оставь такими же):

- `img/logo.png` — логотип в шапке;
- `img/card1.png`, `img/card2.png`, `img/card3.png` — фото на 3 карточках товара;
- `img/slide1.jpg` … `img/slide4.jpg` — 4 фото для слайдера (одинакового размера, напр. 800×360).

Если хочешь другие имена файлов — поменяй путь в `src="img/..."`
(логотип — в `includes/header.php`, карточки — в `index.php`).

---

## 3. Цвета и шрифт

Открой `css/style.css`, самый верх — блок `:root`. Там по одной строке на
каждый цвет (с подписями) и строка `--font` со шрифтом.
Доступные шрифты (лежат локально): **Rubik, Roboto, Oswald, Inter**.

---

## ⚙️ Установка и запуск

1. Положи папку в каталог сайтов (XAMPP `htdocs/demo`, OpenServer `domains/demo`), запусти Apache + MySQL.
2. `http://localhost/phpmyadmin` → «Импорт» → файл `database.sql` → «Вперёд».
3. ER-диаграмма (Модуль 1): база `demo` → вкладка **Designer** (связи `users → applications → reviews`).
4. Открой `http://localhost/demo/`. Доступы:
   - Админ — `Admin26 / Demo20`
   - Тест-пользователь — `ivan123 / parol123`
5. Сценарий: регистрация → вход → заявка → войти админом → сменить статус → отзыв в кабинете.

## 📁 Файлы

```
demo/
├── index.php          # главная: слайдер, название, описание, карточки, контакты, оплата
├── register.php       # регистрация
├── login.php          # вход + уведомления
├── logout.php
├── application.php    # заявка (карточка / дата / время / оплата)
├── cabinet.php        # кабинет: заявки + слайдер + отзывы
├── admin.php          # админка: статусы + фильтр + сортировка + пагинация + тост
├── database.sql       # база данных (3 таблицы со связями)
├── includes/  db.php · auth.php · header.php · footer.php
├── css/  bootstrap.min.css · fonts.css · style.css   (цвета и шрифт — в style.css)
├── fonts/  Rubik · Roboto · Oswald · Inter (локально)
├── js/  bootstrap.bundle.min.js · validation.js
└── img/  logo.png · card1..3.png · slide1..4.jpg   (ЗАМЕНИ на свои!)
```
